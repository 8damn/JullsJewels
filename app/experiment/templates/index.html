Konfigurátorový sandbox

Tři odlišné přístupy ke konfigurátoru šperků. Cílem je porovnat UX z pohledu zákazníka i admina a vybrat z každého to nejlepší. Všechny vizualizéry pracují s fotkami korálků a běží plně klientsky (žádná data se neukládají).

{# V1 #}
V1 — fotorealismus

Photostrand

komplexnost: středníkontrola: výběr barev

Zákazník vidí fotorealistický oblouk náramku. Admin definuje pevný vzorec (sekvenci rolí), zákazník mění jen barvy. Filozofie: "vyber barvy, my složíme."

Zákazník → Admin →
{# V2 #}
V2 — stavebnice

Slotbuilder

komplexnost: vysokákontrola: maximální

Drag-and-drop korálků do prázdných slotů. Symetrické zrcadlení, paleta vlevo, plná kreativita. Admin definuje šablonu (počet slotů, povolené typy). Filozofie: "postav si sám."

Zákazník → Admin →
{# V3 #}
V3 — kurátorství

Quickpicker

komplexnost: nízkákontrola: minimální

3 jednoduché otázky → 4–6 hotových návrhů z předvybraných kolekcí. Admin dělá kreativní práci, zákazník jen vybírá. Filozofie: "klikni co se ti líbí, neřeš detaily."

Zákazník → Admin →
Sandbox je izolovaný od hlavního projektu — žádné DB modely, žádné form submity. Vše je klientský experiment. Hlavní konfigurátor zůstává netknutý na /configurator.
